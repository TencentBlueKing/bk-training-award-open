## 安装与使用

### 安装 {page=#/introduction}

`bkui-cli` 已发布至 [npm](https://www.npmjs.com/package/bkui-cli)，只需要在任意目录下执行以下命令便可以安装 `bkui-cli`。

```bash
$ npm install -g bkui-cli
```

### 使用 {page=#/introduction}

`bkui-cli` 全局安装完毕后，即可输入 `bkui` 全局命令。

```bash
$ bkui
```

![help](../../images/help.png)
